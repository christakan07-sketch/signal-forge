# Evidence Sheets

## Purpose
Evidence sheets capture the actual observations collected for a study.

A study asks the research question.
An evidence sheet records what was observed, what was missing, and whether the case passed or failed the intended test.

## Workflow
1. start from a study file
2. open a matching evidence sheet
3. record observations by time window
4. record confirmation and contradiction
5. decide pass / mixed / fail
6. write lessons and next action
