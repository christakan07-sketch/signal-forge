# Follow-Through Study Template

## Purpose
Check whether the first event signal receives confirmation from later price action, additional headlines, or cross-asset agreement.

## Linked case
- case_id:
- event_family:

## Confirmation channels
- follow-up headlines
- cross-asset confirmation
- sector relative performance
- volume / volatility expansion

## Key questions
- did the market continue in the same direction after the first reaction?
- did another asset confirm the same narrative?
- did confirmation improve or weaken the case?

## Metrics
- direction consistency
- cross-asset agreement score
- follow-up headline count
- persistence versus fade

## Failure modes
- contradictory follow-up news
- weak confirmation
- narrative drift
