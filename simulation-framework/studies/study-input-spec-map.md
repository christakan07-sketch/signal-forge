# Study Input Spec Map

## Persistence study needs
Must have:
- event timestamp
- primary asset prices
- benchmark prices
- volatility measure

Helpful:
- follow-up headlines
- cross-asset snapshots

## Follow-through study needs
Must have:
- event timestamp
- follow-up headlines
- primary asset prices
- confirmation asset prices
- benchmark comparison

Helpful:
- volume changes
- policy confirmation timeline

## False-positive study needs
Must have:
- event timestamp
- initial reaction snapshot
- later reaction snapshot
- contradiction evidence or absence of confirmation

Helpful:
- analyst notes on why the thesis looked good initially

## Cross-asset confirmation study needs
Must have:
- primary asset move
- confirmation asset moves
- common observation windows
- benchmark-relative comparison

Helpful:
- lead-lag notes
- asset-specific catalyst notes
