# Market Intelligence Operator Checklist

## Before starting
- [ ] headline captured
- [ ] source and timestamp recorded
- [ ] source tier assigned

## Scout stage
- [ ] event summary written
- [ ] entities identified
- [ ] event type assigned
- [ ] urgency and novelty noted

## Atlas stage
- [ ] direct assets listed
- [ ] indirect assets listed
- [ ] sectors/themes listed
- [ ] relevance score estimated

## Pulse stage
- [ ] direction estimated
- [ ] horizon estimated
- [ ] confidence noted
- [ ] catalyst vs noise judgment written

## Sage stage (if needed)
- [ ] hypothesis written
- [ ] assumptions listed
- [ ] invalidation conditions listed
- [ ] study type suggested

## Shield stage (if needed)
- [ ] evidence gaps listed
- [ ] risk notes listed
- [ ] decision ladder result chosen
- [ ] next action chosen

## Final
- [ ] main summary prepared
- [ ] case library updated
- [ ] study/evidence stub created if needed
