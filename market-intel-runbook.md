# Market Intelligence Runbook

## Example: one new headline
1. paste headline into intake template
2. assign source tier
3. create normalized event record
4. run scout reasoning
5. run atlas mapping
6. run pulse impact pass
7. decide fast-lane or slow-lane
8. if slow-lane, run sage
9. run shield review
10. write main summary
11. create or update case-library entry
12. create study/evidence stub if research continues

## Fast-lane definition
Use fast-lane when the output is likely to stop at:
- archive
- watch
- basic research note

## Slow-lane definition
Use slow-lane when the output might become:
- a reusable case
- a study candidate
- a simulation candidate
- a high-value research memo

## Operator advice
If uncertain, choose slow-lane.
It is cheaper to do one extra structured review than to build on a weak interpretation.
