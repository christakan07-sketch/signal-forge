# PM Daily Heartbeat

## What to check each day
1. active projects status
2. agents doing what
3. blockers and risks
4. priority suggestions

## Report format
One paragraph + short bullet list for each active project.

## When to escalate
- agent stalled for >1 day
- blocker affecting multiple projects
- missed milestone deadline

## File to update
- `task-tracker.md` (every day)
- `status-reports/YYYY-MM-DD.md` (weekly)
