# Event Types Taxonomy

## Core event types
- energy_supply_shock
- commodity_supply_shock
- geopolitical_risk
- regulatory_action
- monetary_policy_signal
- macro_data_surprise
- company_specific_event
- liquidity_stress
- cyber_or_operational_incident
- policy_announcement

## Notes
- An event can have one primary type and multiple secondary tags.
- Prefer simple, stable labels over overly clever categories.
